# 批次名称默认值配置使用说明

## 配置位置
批次名称的默认格式现在存储在 `config/config.R` 文件中的 `SoyCross$config$cross$default_batch_format` 配置项。

## 修改方法
要更改默认的批次名称格式，请按照以下步骤操作：

1. 打开 `config/config.R` 文件
2. 找到 `default_batch_format` 配置项
3. 修改其值为所需的格式

### 格式说明
- `%Y` - 四位年份 (例如: 2026)
- `%y` - 两位年份 (例如: 26)
- 其他字符将按字面意思显示

### 示例格式
```r
# 春季批次（默认）
default_batch_format = "%Y春季"

# 夏季批次
default_batch_format = "%Y夏季"

# 年份+季节数字
default_batch_format = "%Y01"

# 年份+字母标识
default_batch_format = "%Y-A"

# 更复杂的格式
default_batch_format = "G%y春季"
```

## 生效方式
配置更改后，下次启动应用程序时会自动加载新的默认值。

## 应用范围
此配置会影响以下位置的批次名称默认值：
- 杂交配置模块的组合矩阵页面
- 其他使用默认批次名称格式的地方

# 账本生成参数配置

## 配置位置
账本生成的各个参数默认值现在存储在 `config/config.R` 文件中的 `SoyCross$config$field` 配置组中。

## 配置项说明
- `default_prefix` - 组合前缀格式 (默认: "G%y")
- `default_start_n` - 起始编号 (默认: 1)
- `default_digits` - 编号位数 (默认: 3)
- `default_place` - 种植地点 (默认: "海南")
- `default_rows` - 种植行数 (默认: 2)
- `default_rp` - 重复数 (默认: 1)
- `default_interval` - 重复数间隔 (默认: 999)

## 修改方法
要更改账本生成的默认参数，请打开 `config/config.R` 文件并修改 `field` 部分的相应值。

### 示例格式
```r
field = list(
  default_prefix = "W%y",  # 武汉项目前缀
  default_start_n = 1,        # 起始编号
  default_digits = 4,         # 使用4位编号
  default_place = "武汉",     # 更改默认地点
  default_rows = 3,           # 3行种植
  default_rp = 2,             # 2个重复
  default_interval = 500      # 间隔设为500
),
cross_matrix = list(
  default_memo = "杂交组合"    # 组合特点默认值
),
```
