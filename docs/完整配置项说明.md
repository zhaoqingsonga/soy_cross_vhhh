# 完整配置项说明

## 配置文件位置
所有配置项均位于 `config/config.R` 文件中，通过 `SoyCross$config` 对象访问。

## 配置项分类

### 1. 杂交配置 (cross)
- `default_batch_format` - 批次名称默认格式 (默认: "N%y11")

### 2. 字段配置 (field)
- `default_prefix` - 账本生成组合前缀 (默认: "N%y11")
- `default_start_n` - 起始编号 (默认: 1)
- `default_digits` - 编号位数 (默认: 3)
- `default_place` - 种植地点 (默认: "武汉")
- `default_rows` - 种植行数 (默认: 2)
- `default_rp` - 重复数 (默认: 1)
- `default_interval` - 间隔 (默认: 999)

### 3. 组合矩阵配置 (cross_matrix)
- `default_memo` - 组合特点默认值 (默认: "杂交组合")

## 配置示例
```r
SoyCross$config <- list(
  paths = list(
    db_path = "data/db/soy_cross.db",
    backup_dir = "data/db"
  ),
  sqlite = list(
    busy_timeout_ms = 10000,
    enable_foreign_keys = TRUE
  ),
  cross = list(
    default_batch_format = "N%y11"
  ),
  field = list(
    default_prefix = "N%y11",  # 默认组合前缀
    default_start_n = 1,       # 起启编号
    default_digits = 3,        # 编号位数
    default_place = "武汉",    # 默认地点
    default_rows = 2,          # 默认行数
    default_rp = 1,            # 默认重复数
    default_interval = 999     # 默认间隔
  ),
  cross_matrix = list(
    default_memo = "杂交组合"  # 组合特点默认值
  ),
  analysis = list(),
  io = list(),
  safety = list()
)
```

## 修改配置
要修改任何配置项，只需编辑 `config/config.R` 文件中对应的部分，保存后重新启动应用程序即可生效。

## 注意事项
- 修改配置后需要重启应用程序才能生效
- 请确保配置值符合预期的数据类型
- 在生产环境中修改配置前，建议先在测试环境中验证
