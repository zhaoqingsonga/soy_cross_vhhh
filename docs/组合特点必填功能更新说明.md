# 组合特点必填功能更新说明

## 更新概述
将杂交配置模块中的组合特点字段从"可选"改为"必填"，以确保每个杂交组合都有明确的特点描述。

## 修改文件

### 1. R/mod_cross_app.R
- 将UI中的组合特点输入框标签从"组合特点（可选）"改为"组合特点（必填）"
- 添加placeholder提示"请输入组合特点"
- 在服务器端逻辑中添加验证，确保memo字段不为空
- 如果memo字段为空，显示错误通知"组合特点不能为空，请填写组合特点"

### 2. apps/run_cross_app.R
- 添加组合特点输入框到UI，设置为必填项
- 在服务器端逻辑中添加验证，确保memo字段不为空
- 更新create_specific_cross_plan函数调用，传递memo参数

## 技术实现

### 验证逻辑
```r
# 验证 memo 不为空
if (!nzchar(input$memo)) {
  showNotification("组合特点不能为空，请填写组合特点", type = "error")
  return()
}
```

### 函数调用更新
- `create_specific_cross_plan` 函数现在接收 memo 参数
- memo 值存储在数据库的 crosses 表中

## 用户体验改进
1. 更好的数据完整性：确保每个杂交组合都有特点描述
2. 清晰的错误提示：当用户忘记填写组合特点时提供明确的错误信息
3. 一致的界面：所有版本的应用程序都使用相同的必填验证逻辑

## 向后兼容性
- 此更改不影响现有数据库中的数据
- 所有现有功能保持不变
- 只是新增了验证逻辑，强制用户填写组合特点

## 配置文件
- 该功能与配置文件中的其他设置保持一致
- 默认值和格式遵循现有配置模式
- 组合特点的默认值可在 config/config.R 文件的 cross_matrix.default_memo 配置项中修改
